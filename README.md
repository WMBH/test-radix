## Тестовое задание для Radix

**Demo:**

http://text-radix.surge.sh/

**Stack:**

- ReactJS
- Redux
- Axios
- Lodash
- Bootstrap (React)
- redux-form
- uuid

**Features:**

- Поиск по имени и названию миссии
- Сортировка по всем полям таблицы(клик по заголовку)
- Добавление/Удаление элементов
- Пагинация
